## Recording Tips

### Equipment Setup
- **Screen Recording**: Use scrcpy for Android mirroring or Android Studio screen recording
- **Alternative**: Use built-in Android screen recorder
- **Audio**: Clear microphone, quiet environment
- **Resolution**: 1080x1920 (portrait) or mirror to desktop at 1920x1080

### Before Recording
1. Test all features to ensure they work
2. Prepare demo data (friends, trips, messages)
3. Clear notifications
4. Charge devices
5. Practice timing (aim for 4:30-5:00)

### During Recording
- Speak clearly and confidently
- Don't rush through features
- Show smooth transitions
- Highlight unique features
- Keep mouse movements deliberate

### Editing
- Add transitions between sections
- Include text overlays for key points
- Add background music (optional, keep low)
- Include title card and end card
- Export at 1080p, upload to YouTube
